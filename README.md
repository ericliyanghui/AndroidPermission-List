# AndroidPermission-List
安卓权限列表（更新）  Android Permission List（Updating）

因研究需要，将Android到目前为止的所有官方Permission列出来，虽然也用Android SDK 28的getName()进行罗列，但发现很多与官方不符，交集下选了官方的。
如有遗漏，欢迎指正。
